Criando o projeto

Para criar o projeto é bem simples, vamos utilizar o comando abaixo
ionic start NomeDoSeuApp tabs --type=angular

Logo em seguida digita o seguinte comando no prompet
cd NomeDoSeuApp code .
Code irá abrir o visual studio code na pasta do projeto

Um ptojeto será criado com uma estrutura semelhante a esta


Rodadndo o projeto

Para rodar o projeto, digite no prompet de comando
ionic serve
